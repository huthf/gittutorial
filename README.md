# A Recipe for Guacamole

## Ingredients

- avocados
- onions
- salt
- pepper
- tomatos

## Instructions

- chop onion and tomatoes
- cut open avocado and remove stone
- scoop out flesh
- Put into a bowl and mix.
- Make sure to enjoy!
